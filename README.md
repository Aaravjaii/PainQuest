# PainQuest 🏥🎮

PainQuest is a Unity-based gamified fitness and rehabilitation application that helps users perform pain-relief exercises through guided sessions, posture analysis, and an XP-based progression system.

## Features
- Guided exercise sessions
- AI-based posture tracking
- Real-time scoring and feedback
- XP & Level System
- Multiple pain categories
- Interactive and user-friendly UI

## Tech Stack
- Unity 6
- C#
- Unity Sentis
- OpenCV
- TensorFlow Lite
